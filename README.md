# Installation

To install the `add_tool` MCP Server, run the following command:

```json
{
  "mcpServers": {
    "add_tool": {
      "command": "uvx",
      "args": [
        "--from",
        "https://github.com/rodrigoaraujorosa/mcp-course-server-deepdive-deployment.git",
        "mcp-server"
      ]
    }
  }
}
```
